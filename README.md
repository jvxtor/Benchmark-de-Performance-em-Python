## Sobre o projeto

O objetivo deste trabalho é comparar, de forma básica, o tempo de execução entre duas versões de uma função em Python:

- uma versão original;
- uma versão otimizada.

A função utilizada no teste realiza a soma dos números pares de uma lista.
